## How to Get Listed

If you discover a security vulnerability in Honua:

1. Report it to **security@honua.io**
2. Wait for our acknowledgment (within 48 hours)
3. Work with us to verify and fix the issue
4. After the fix is deployed, we'll add you here (with your permission)

### What We List

- Your name or handle (as you prefer)
- Month/year of discovery
- Brief description of the vulnerability class
- Optional: Link to your website, GitHub, or Twitter

We **do not** list:
- Specific vulnerability details (until responsible disclosure period ends)
- Exploits or proof-of-concept code
- Information that could aid attackers
